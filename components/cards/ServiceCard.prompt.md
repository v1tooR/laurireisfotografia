The signature Lauri Reis card — a portfolio photograph with navy scrim, gold eyebrow and serif title. Description fades in on hover. Use for the services grid and portfolio category links.

```jsx
<ServiceCard
  image="assets/imagery/15-anos.jpg"
  eyebrow="Debutante"
  title="15 Anos"
  description="A festa, o vestido, o brilho — registrados do ensaio à pista."
  href="#15-anos"
/>
```

Default ratio is portrait `3 / 4`. Place several in a CSS grid with `gap: 24px`. Always pass real portfolio imagery; never a flat color.
